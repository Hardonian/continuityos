# Lawful public data source map for a Canadian continuity / national-security decision-support system

**Scope.** Public, lawful, unclassified sources suitable for resilience, continuity, hazard, logistics, infrastructure, and open-source situational awareness. Emphasis is on feeds active in 2025–current and on free, keyless, or readily obtainable access. This is a source map, not an endorsement of operational reliance. Verify current terms, schemas, service health, and data currency before production use.

**Access labels:** `None` = no API key observed; still respect user-agent, rate, attribution, and terms. `Registration` = account/token obtainable without a commercial contract. `Key` = subscription/API key required. `Licensed` = free but an explicit licence/terms acceptance applies.

## Recommended minimum architecture

1. **Authoritative alert lane (minutes):** ECCC CAP/GeoMet, DFO IWLS, NOAA SWPC/NWS, provincial emergency feeds where available. Cache raw payloads, signatures/ETags, issuance/expiry, and QC flags.
2. **Operational-environment lane (hours–days):** CWFIS/Fire M3, hydrometric observations, Sentinel-1/2, RCM/Landsat, marine activity layers. Preserve acquisition time separately from publication time.
3. **Continuity/economy lane (weekly–monthly):** TDIH, StatCan WDS/trade, CER/CCEI energy, ISED/CRTC broadband, UN Comtrade. Track revisions and release calendars.
4. **Open-source context lane (event-driven):** NATO/NORAD official releases, GDELT, ACLED, GFW. Treat as leads/indicators, never as sole evidence for a security decision.
5. **Resilience controls:** source allow-list; per-source terms/attribution registry; rate-limit/backoff; schema-contract tests; freshness SLAs; provenance and hash of every raw response; uncertainty/QC propagation; dual-source corroboration for high-consequence alerts; local last-known-good cache and degraded-mode operation.

## Legal and analytical cautions

- “Public” does not mean unrestricted: GFW is non-commercial/public-good by default; CHS/IWLS and ECCC Datamart have explicit licences; UN Comtrade, NASA DONKI, ONC, ACLED and Copernicus services may require registration, keys, quotas, or extra terms.
- Avoid using public sources to infer protected personal information, sensitive facility vulnerabilities, or non-public military operations. Keep collection to lawful public records and published aggregate data.
- AIS is not a complete maritime picture; satellite imagery is not continuous surveillance; news/event datasets are not verified intelligence. Preserve source caveats in the decision record.
- Validate current endpoints and licence text before deployment; agencies can change URLs, schemas, quotas and access policies.
